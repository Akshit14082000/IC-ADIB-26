# IC ADIB-26 (International Conference)

Welcome to the official website repository for the **IC ADIB-26** conference. 

This website serves as the informational hub for the International Conference, organized by the **Department of Management Studies & Industrial Engineering** at **Indian Institute of Technology (Indian School of Mines) Dhanbad**. The conference is scheduled to take place on **18th – 19th September 2026** as part of the institute's centenary celebrations.

## What is this Website For?

This website provides all essential details about the conference for prospective authors, researchers, delegates, and industry participants. Key features include:

- **Conference Information**: Details regarding the theme, tracks, and objectives of the conference.
- **Important Dates**: A timeline outlining the submission, acceptance, and registration deadlines.
- **Submissions & Registration**: Easy access to submit research abstracts and register for the event.
- **Travel & Accommodation**: A comprehensive guide on how to reach the IIT(ISM) Dhanbad campus, along with a curated directory of nearby hotels, on-campus food joints, and local places to explore.
- **Downloads**: Quick links to download the official conference brochure, submission guidelines, and conference flyer.
- **Committee Details**: Information about the organizing team and committee members.

## Structure

- `index.html`: The main homepage containing conference details, tracks, timelines, and committee information.
- `register.html`: Dedicated page outlining the registration fees and payment instructions.
- `travel.html`: An extensive travel guide to assist delegates with accommodation, transportation, and local tourism.
- `chatbot.js` / `chatbot.css`: Custom interactive assistant embedded into the site.
- `images/`: Stores all graphics, logos, and UI assets.
- `pdf/`: Contains downloadable documents such as the brochure, flyer, and guidelines.

## Development

The frontend is built with clean, modern HTML, CSS, and vanilla JavaScript. It is designed to be fully responsive for a premium experience on both desktop and mobile devices.
